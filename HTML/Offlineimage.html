Leider gab es ein Problem mit dem Stream:

Stream ist offline!

    :(


Zur Lösung des Problems können folgende Schritte versucht werden:
1. Stream durch drücken von F5 neu laden
2. Rage Quit mittels Alt+F4
3. Stream per Taskmanager beenden (Strg+Umschalt+ESC)
4. Gerät aus dem Window werfen

Wenn das Problem trotzdem weiterhin bestehen sollte,
kann das Gerät ausgeschaltet werden.
(Kann auch vor dem letzten Schritt getan werden!)

Auf https://www.twitch.tv/develeon64 kann
der aktuelle Status abgerufen werden.
Sobald das Problem behoben wurde (aktualisieren des Status mit F5),
kann das Gerät wieder eingeschaltet und wie gewohnt genutzt werden.
